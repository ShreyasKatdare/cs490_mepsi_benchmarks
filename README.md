# MEPSI Code

This is the implementation for 'MEPSI: An MDL-based Ensemble Pruning Approach with Structural Information'. This package is based on Python and Cython.

## How to Install
Run the following command to install the mepsi package

```bash
pip install numpy
pip install --upgrade cython
python setup.py install
```

## How to Run Tree-based Mepsi Algo

Run the following command:

```bash
python examples/main.py
```
